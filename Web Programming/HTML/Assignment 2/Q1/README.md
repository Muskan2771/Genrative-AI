Assignment 2 – Question 1
HTML Links and Images

Objective:
The objective of this assignment is to demonstrate the use of anchor (<a>) and
image (<img>) tags along with their important attributes in HTML.

Technologies Used:
- HTML5

HTML Tags Used:

Anchor Tag (<a>):
Attributes used:
- href : Specifies the link URL
- target : Specifies where to open the linked document
  - _self  → Opens link in the same tab
  - _blank → Opens link in a new tab

Image Tag (<img>):
Attributes used:
- src    : Specifies image source
- alt    : Provides meaningful alternative text
- height : Sets image height
- width  : Sets image width

Images Used:
- Internet images from Unsplash
- Meaningful alt text provided for accessibility

Links Included:
1. Link that opens in a new tab using:
   target="_blank"

2. Link that opens in the same tab using:
   target="_self"

Assignment Requirements Covered:
✔ Used <a> tag with href and target attributes  
✔ Included two links (same tab & new tab)  
✔ Used <img> tag with src, alt, height, and width  
✔ Added meaningful alt text for images  

Output:
The HTML page displays images along with clickable links.
One link opens in the same browser tab, and the other opens in a new tab,
fulfilling all assignment requirements.
